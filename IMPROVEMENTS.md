# Cross-Idea Improvements

Track rollouts of the same improvement across multiple idea stacks.

This file is updated automatically when you run:
```sh
npm --prefix cdk run idea:rollout -- --improvement="<name>"
```
Each rollout entry records the improvement label, the stages it was deployed to, the commit SHA, and the deploy timestamp. See `IDEAS.md` for the per-idea status and `ideas/<idea-id>/IMPROVEMENTS.md` for per-idea rollout history.
